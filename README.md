# 元素週期表互動式學習（GitHub Pages 版）

## 內容特色
- 中文元素週期表，英文縮寫顯示在中文名稱下方
- 點擊元素後，顯示：
  - 原子序
  - 原子量
  - 週期 / 族群
  - 課堂重點
  - 常見應用
  - 常見化合物
- 可搜尋元素名稱、英文名稱、符號、原子序
- 可依元素族群篩選
- 附簡單隨機測驗

## 檔案說明
- `index.html`：主頁面
- `style.css`：版面與色彩
- `data.js`：元素資料
- `script.js`：互動功能

## 如何放到 GitHub Pages
1. 在 GitHub 建立新的 repository。
2. 將這四個檔案上傳到 repository 根目錄。
3. 到 **Settings → Pages**。
4. 在 **Build and deployment** 中選擇：
   - Source：`Deploy from a branch`
   - Branch：`main` / `(root)`
5. 儲存後，等待 GitHub 產生網址。
6. 開啟 GitHub Pages 網址即可使用。
